从零开始：如何搭建你的个人博客

2024-03-20 技术 阅读时间：5分钟 258

前言

搭建个人博客是一个展示自己技术能力和分享知识的好方式。本文将介绍如何使用 GitHub Pages 搭建个人博客。

准备工作

在开始之前，你需要准备以下工具：

  • GitHub 账号
  • 基础的 HTML/CSS 知识
  • 文本编辑器（如 VS Code）

具体步骤

1. 创建 GitHub 仓库

首先，我们需要在 GitHub 上创建一个新的仓库。仓库名格式为：username.github.io


git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/username/username.github.io.git
git push -u origin main

2. 设置 GitHub Pages

在仓库设置中启用 GitHub Pages 功能：

  1. 进入仓库设置
  2. 找到 Pages 选项
  3. 选择部署分支
  4. 保存设置

总结

通过以上步骤，你就可以成功搭建自己的个人博客了。接下来，你可以：

  • 自定义博客主题
  • 添加更多功能
  • 开始写作分享
